
# 🎨 Theme Editor

使用 React 实现的基于 CSS 变量的主题编辑器

## 功能

- 导入/导出 `CSS` 变量文件
- 导入/导出 `JSON` 文件
- 所见即所得，改动实时预览

## 本地运行

克隆本项目

```bash
  git clone https://github.com/ihccc-cn/theme-editor.git
```

进入文件夹

```bash
  cd theme-editor
```

安装依赖

```bash
  pnpm install
```

启动本地服务

```bash
  pnpm run dev
```
